## Getting started
### Install necessary packages
```shell
npm install
cd frontend
npm i
```

### Add the .env file to root directory
Ask team members for .env credentials

### Run backend server (in root directory)
```shell
node server.js
```

### Run frontend server
```shell
cd frontend
npm start
```
